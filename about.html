About Project

This project is a mini project on the ideaology of having a website through which a user like student or a working professional can find out or can be said that they can select a desired resume template for thier resumes in more convinient and faster way. this project integrate different resume templates from different template making websites.Some of those websites are ADOBE,Resume Builder,Enhancy,live carrer nad many more.The purpose of this project is to get all the good n famous resume templates that are present on different website at one single place. This website is purely a Front-End website,which is made by the use of HTML and CSS totally. Futher in future there can be a possibilities of uploading self made templateusing JS n other web dev technologies.
